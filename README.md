# Sistema de Transporte Inteligente

# Descripción
Este proyecto simula la gestión de una flota de vehículos en una ciudad inteligente utilizando POO en Java

# Tecnologías
* Java
* UML (draw.io)
* GitHub

# Estructura
* Clase abstracta: Vehiculo
* Interfaz: IConectable
* Clases concretas: DronTransporte, CamionAutonomo
* Clase de control: CentroControl

# Funcionalidades
* Registro de vehículos
* Monitoreo de flota
* Aplicación de polimorfismo
